# Pointy Rocket V1

## Introduction
### Pointy Rocket (V1) is an actively controlled model scale rocket developed by the Engineering Society of Castleknock College. The rocket uses a conventional aerospace technique, known as thrust vectoring, to stablize the rocket. 

## PROJECT COMPONENTS:
1. Avionics System
2. TVC
3. Airframe
4. Simulation
5. Documentation
6. Software/Firmware
